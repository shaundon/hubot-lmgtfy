hubot-lmgtfy
============

Allows Hubot to create LMGTFY queries.

## Installation

Pop `lmgtfy.coffee` into your Hubot's `scripts` folder.

## Usage

`hubot lmgtfy <terms>`.

## Behind The Scenes

Works entirely offline - simply URL encodes the terms, and appends them to a LMGTFY URL.
